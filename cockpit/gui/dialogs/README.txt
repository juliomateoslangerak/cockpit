This directory contains code related to displaying dialogs to the user.

experiment: This directory contains code related to experiment dialogs.

enumerateSitesPanel.py: A sub-dialog widget for letting the user select 
  previously-marked sites. 
getNumberDialog.py: A simple dialog to prompt the user for a number.
gridSitesDialog.py: Generate a rectangular array of regularly-spaced marked
  sites. 
offsetSitesDialog.py: Adjust the positions of marked sites. 
safetyMinDialog.py: Set the Z safety minimum.
