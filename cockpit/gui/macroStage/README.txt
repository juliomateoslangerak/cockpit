The Macro Stage window gives the user a high-level overview of where they are
with respect to the motion limits of their stage. 

macroStageBase.py: A base class for the XY and Z stage displays; includes code
  for text display, scaling, and the like.
macroStageWindow.py: The window UI code, including the buttons along the 
  bottom. 
macroStageXY.py: Displays the XY stage position.
macroStageZ.py: Displays the Z stage position, including a histogram of past
  experiment altitudes.
